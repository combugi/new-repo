preference(ctrl+k)

-일러스트레이터 만의 vector방식의 특징
-얼라이싱(깨짐) 현상이 발생하지 않는다
-파일용량이 적다(사진같은 이미지 제외)
-도형 오브젝트 위주로 작업이 이루어 진다.
-도형은 점,선(edge;가상 선),면 으로 이루어진다.
-도형을 제작할시 선(stroke 눈에 보이는 선) 과 면(shape) 으로 구성  된다

오브젝트의 생성
-도형툴 선택후 드래그를 이용
-도형툴 선택후 클릭(세부 옵션 창)
-주의사항 (선택해제 후 생성,선택 툴클릭후 빈곳 클릭 또는 ctrl 빈곳 클릭)

선택툴
/전체선택툴(v)
-오브젝트 전체를 선택
-복사 alt + 드래그
-정렬,정돈
객체를 선택후 move tool을 선택


/직접선택툴(a)
-점,선을 선택
그룹,언그룹(컨트롤 g 컨트롤 쉬프트 g)

유용한 기능
●ctrl+y=와이어프레임 보기
●ctrl+shift+alt+s 다양한 이미지 포멧으로 저장
●show grid (ctrl+'),snap to grid( ctrl+ shift+')
●동일한 위치에 객체를 복사 붙여넣기 컨트롤 씨 이후 컨트롤 에프
●마술봉 동일한 색상과 스트록 선택
●선일 경우는 전체선택툴로 선택후 delete를 하면 반절된다


■펜기능(p)
  
●드로잉 도구
-클릭을 직선 오브젝트를 생성
-드래그를 이용해서 곡선 오브젝트를 생성
-새로운 드로잉 시작시 컨트롤 키를 이용해 빈곳 클릭
●오브젝트 편집
-점 위를 클릭:점을 제거(직접선택툴로 점을 제거시에는 점과 연결된 선까지 제거)
-선 위를 클릭:점을 생성
-alt+점클릭:베이지어 제거
-alt+점위에서 드래그:베이지어 생성
-펜툴 사용시 ctrl키를 누르고 있는 동안은 직접선택툴와 동일한기능

■shapemode와 pathfinder(ctrl+shift+f9)
	-객체를 선택후 
●shapemode
-두개 이상의 오브잭트를 최종하나로 만든다

●pathfinder
