이미지출력하기

경로

  • 절대경로
    -루트경로부터 시작하는 방식 - 도메인 절대경로: http로 시작하는 도메인 경로 - 서버 절대경로: /로 시작하는 방식
  • 상대경로
    -작성하는 문서 기준으로 부터 문서를 찾는 방식
    -현재폴더:./ -상위폴더:../

img요소

  • 웹접근성을 준수하기 위해서 alt(대체문자) 속성을 추가해야함
  • heading요소 안에는 img요소만 존재해서는 안됨
  • ratio(비율)을 유지하기 위해서 넓이나 높이 하나만 설정
  • ratio(비율)이 변할경우 object fit 속성과 같이 사용

background 속성

심미성을 위해 사용하기 권장

배경이미지 출력하기

  • 배경이미지는 패턴형태로 출력된다.
  • background-repeat 속성 (repeat,no-repeat,repeat-x,repeat-y)
  • background-position 속성(가로x축:left center right,세로y축:top center bottom)

배경색과 배경이미지를 같이 사용하기

배경이미지 크기

  • background-size
    cover:객체를 빈틈없이 배경으로 채우기
    container:객체안에 모든 배경이 적용되기
