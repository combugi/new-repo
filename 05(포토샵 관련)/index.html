anchor요소

hyper link를 구현하는 목적으로 사용

외부링크(external link)

  • 하이퍼 링크를 적용할 목적으로 사용
  • 하이퍼링크를 적용하려면 href(hyper reference)속성을 추가
  • 타겟으로 브라우저 윈도우를 제어 (_self, _blank)
  • anchor객체는 기본적으로 agent

    • 내부링크(책갈피,internal link)

      동일 웹페이지 내에서 위치 이동

    • a요소를 이용해 위치와 이동버튼을 설계
    • 위치표식은 name속성,이동버튼은 href속성을 사용
    • href속성의 #(임시링크)은 현재 페이지를 의미함
