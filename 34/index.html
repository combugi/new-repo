3.justify-content

  • flex컨테이너에 적용,flex컨테이너 자식들의 위치분할
  • flex 컨테이너에 적용
  • flex-start:기본값
    flex-directon이row일경우 flex start왼쪽
    flex-directon이coulmn일경우 flex start위쪽
    flex-directon이row-reverse일경우 flex start오른쪽 역순
    flex-directon이coulmn-reverse일경우 flex start아래쪽 역순
  • space-between:부모컨테이너의 공간을 자식들이 분배해 갖는다
    flex-direction이 row일 경우 좌우분할
    flex-direction이 coulmn일 경우 상하분할
  • flex-end:반대쪽으로 자식들을 정렬한다
    flex-direction이 row일 경우 자식들을 오른쪽을 정렬
    flex-direction이 coulmn일 경우 자식들을 아래쪽을 정렬
    flex-direction이 row-reverse일 경우 자식들을 왼쪽으로 역순 정렬
    flex-direction이 coulmn-reverse일 경우 자식들을 위쪽을 역순 정렬

4.align-items

  • strech
    flex-direction이 row일 경우 자식들의 높이가 부모의 높이가 같아진다
    flex-direction이 column일 경우 자식들의 넓이가 부모의 높이가 같아진다
  • center flex-direction이 row일 경우 자식들의 원래 높이가 되며 부모의 수직중앙으로같아진다
    flex-direction이 column일 경우 자식들의 원래 넓이가 되며 부모의 수평중앙으로같아진다
  • :flex-start
    flex-direction이 row일 경우 자식들의 원래 높이가 되며 윗쪽 정렬 flex-direction이 column일 경우 자식들의 원래 넓이가 되며 왼쪽정렬
  • :flex-end
    flex-direction이 row일 경우 자식들의 원래 높이가 되며 오른쪽 정렬 flex-direction이 column일 경우 자식들의 원래 넓이가 되며 아래쪽 정렬
