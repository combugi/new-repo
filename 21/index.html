form요소

사용자가 입력 변결할수 있는 모든 요소

  • 데이터의 전송 범위를 form요소(block)를 설정한다
  • fieldset:form의 정보범위를 구분
  • legend:fieldset의 제목
  • label:폼 요소에 대한 정보 제공
  • input
    • 직접입력 type ()가 속성을 더 넣을 수있다
      • text(size,maxlength,placeholder)
      • passward(size,maxlength,placeholder)
      • number(placeholder,min,max)
      • email(size,maxlength,placeholder)
      • tel(size,maxlength,placeholder)
    • 선택 type
      • radio(name:라디오 버튼일 경우에만 기능이 달라진다;checked:미리 체크를 할 수 있다.)
      • checkbox(checked)
      • date
      • file(accept)
    • 버튼타입
      • submit :데이터 전송기능(새로고침 까지 해주는 기능)
      • reset : 폼요소 초기화(단순 지우는 기능)
      • button : 자바스크립트와 연동해야하는 아무기능 없는 버튼
  • select,option옵션(option) 객체에서만 label 속성 을 쓸수 있다
  • textarea: 태그안의 여백을 생성하지 말것
