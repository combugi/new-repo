cascading style sheet 약자로 웹문서의 디자인을 처리하는 언어임

특징과 주의 사항

  • 우선순위가 존재함
    인라인 스타일>id선택자>클래스 선택자>자손 > 부모
  • 상속성(inherit)을 가지고 있음(상속이 되는 속성과 되지않는 속성이 존재함)
  • head 요소안에서 style요소를 선언 후 선택자{속성:값}형태로 작성한다.

선택자(selector)

  • type selector:
  • 태그선택자(type selector)
    태그명{속성:속성값}
  • 클래스 선택자(class selector)
    .클래스 작명{속성:속성값;}로 방법으로 선언후 html 요소에서 calss속성 으로 호출을한다.반복호출(재사용)이 가능하고 수정관리가 용이함.
  • ID selector
    #아이디 이름{속성:속성값;}로 방법으로 선언후 html 요소에서 id속성 으로 호출을한다.상호간의 충돌방지 차원에서 사용 동일한 아이디는 사용할 수 없음
  • inline style
    html 요소의 style속성을 추가해서 한줄로 속성과 값을 설정
    수정관리가 매우 비효율적임 자바 스크립트를 이용해 css를 변경할 경우 사용됨
