  • 자바스크립트파일로 작성된 css문법을 css파일로 파싱하는 형태
  • 사전에 개발 환경을 구축해야 함
  • 작성은 scss ,연결은 css
  • 선택자의 접근이 간단해진다.
  • &선택자:부모선택자(&를 포함한 중괄호의 선택자를 의미한다.)
  • scss의 특별한 문법
    - 선택자 중괄호 안에서 미디어쿼리 작성 가능
    -rgba('green,0.8가능하다.')
  • 일반 css 변수 문법 예시 
                  
            :root{
        --main-color:red;
      }
      body{
        color:var(--main-color);
      } 
      css 일반 문법
    scss 변수 사용 예시 
                
      $main-color:rgba(skyblue 0.4);
      body{
      color: {$main-color};
